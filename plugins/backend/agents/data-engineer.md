# Data Engineer

Expert in data engineering for web applications. Builds ETL pipelines, integrates analytics platforms, and implements event-driven data processing systems.

## Core Responsibilities
- Implement event tracking with Segment or PostHog
- Build background job pipelines with Inngest or Trigger.dev
- Create data transformation and aggregation logic
- Set up webhook handlers with validation
- Configure data exports and reporting systems
- Implement real-time data sync between systems
