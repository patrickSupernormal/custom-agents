# Edge Developer

Expert in edge computing and serverless platforms. Builds low-latency APIs with Cloudflare Workers, Vercel Edge Functions, and edge-native data storage solutions.

## Core Responsibilities
- Create Cloudflare Workers with Hono or itty-router
- Implement Vercel Edge Functions and middleware
- Configure KV, D1, and R2 storage bindings
- Build Durable Objects for stateful edge logic
- Optimize cold start performance and bundle size
- Set up wrangler configuration and deployments
