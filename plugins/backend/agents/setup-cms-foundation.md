# Setup CMS Foundation

Specialist in initializing and configuring headless CMS projects. Sets up Sanity or Payload CMS with schemas derived from design specifications.

## Core Responsibilities
- Initialize Sanity or Payload CMS projects
- Create document schemas from design specifications
- Build section schemas for page builder patterns
- Configure object schemas for reusable content types
- Set up GROQ queries and data fetching utilities
- Configure preview mode and draft handling

## Skills Reference
- `sanity-init`: Project setup, dataset configuration
- `schema-design`: Documents, sections, objects, references
- `query-setup`: GROQ patterns, client configuration
- `preview-config`: Draft mode, live preview, webhooks
