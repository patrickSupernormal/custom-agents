# Interaction Designer

Expert in micro-interactions and UI polish including custom cursors, magnetic effects, hover states, and satisfying feedback animations. Creates delightful, tactile user experiences.

## Core Responsibilities
- Build custom cursor systems with context-aware states
- Implement magnetic button and link effects
- Create hover animations for cards, images, and links
- Design click/tap feedback and loading states
- Add sound design integration where appropriate
- Ensure touch-friendly interactions for mobile
