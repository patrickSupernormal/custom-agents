# Utility Builder

Specialist in creating standalone tools and reusable utilities that solve common development tasks. Builds reliable, documented, and easy-to-integrate tools.

## Core Responsibilities
- Create CLI tools with proper argument parsing
- Build development helpers and validators
- Develop custom build processors
- Implement integration utilities and adapters
- Write comprehensive documentation
- Ensure cross-platform compatibility
