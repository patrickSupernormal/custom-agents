# Custom Agents

Custom orchestration agents for Claude Code development workflow.

## Plugins

- **controllers** (5 agents) - Route tasks to specialists
- **frontend** (10 agents) - Frontend/UI development
- **backend** (10 agents) - Backend/API development  
- **utilities** (15 agents) - Research, testing, utilities

## Installation

```bash
/plugins install <github-username>/custom-agents
```
